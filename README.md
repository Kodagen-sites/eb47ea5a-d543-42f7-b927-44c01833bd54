# SaharaOps — Cinematic Consulting Site

Operations-consulting landing site for **SaharaOps**, generated by the
cinematic-site-generator skill.

## Stack

- **Next.js 16** (App Router) · React 19 · TypeScript
- **Tailwind CSS 3** · framer-motion 12 · GSAP ScrollTrigger
- Fonts: Fraunces (display) · Inter (body) · JetBrains Mono (mono)

## Homepage section sequence (Archetype G)

1. Hero — ScrollCanvas scroll-scrubbed cinematic (8s Veo clip → 192 frames)
2. Value proposition — oversized type, word reveal
3. Services — 6 stagger-revealed, tilt-reactive cards with parallax thumbnails
4. Showcase — image mosaic gallery (dark contrast band)
5. Type statement — mission line on pale accent ground
6. Process — pinned split-scroll, four-step method
7. Stats — three animated counters (dark band)
8. Testimonials — three quote cards
9. Operating-principles marquee (dark band)
10. CTA — canvas atmosphere + magnetic button

Section background tones rotate across white / mist / ink-dark / accent-soft
(≥4 distinct tones — rule 33.1).

## Assets

Generated with Google Gemini:

- **Hero video** — Veo 3.1 Lite, 8s architectural dolly → `public/frames/frame-0001…0192.jpg`
- **14 stills** — Nano Banana (`gemini-3.1-flash-image-preview`): 6 service
  thumbnails, 4 showcase images, 1 process image, 3 inner-page heroes

Regenerate with `npm run gen:all` (requires `GEMINI_API_KEY`).

## Booking

"Book a Consultation" routes to `/contact`, which embeds a Calendly inline
scheduler. Replace `contact.calendlyUrl` in `content/site-config.ts` with the
real Calendly / Cal.com link.

## Develop

```bash
npm install
npm run dev      # http://localhost:3000
npm run build    # production build (next build --debug-prerender)
npm start
```
